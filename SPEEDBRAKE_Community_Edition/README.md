# ML_F16 Speedbrake Community Edition

Community Edition placeholder for the F-16C Speedbrake Indicator.

This module is in preparation.

## Planned Target Hardware

- Arduino Nano / ATmega328P
- Small OLED or TFT display
- DCS-BIOS / DCS-BIOS Bridge
- DCS F-16C speedbrake position data

## Planned Features

- Startup splash
- Closed / partial / open speedbrake display states
- DCS-BIOS live mode
- Stale-data indication
- Local test mode
- Basic diagnostics
