# ML_F16 AOA Indicator - ESP32-D Community Release

Community tutorial build for an F-16C Angle of Attack tape indicator using:

- ESP32-D / ESP-WROOM-32
- 2.4 inch SPI TFT display
- DCS-BIOS
- F_16C_50_AOA_VALUE

This release is a stripped community version. It does not include PanelReporter, PFLD, fleet reporting, pro build flags, or internal ML_F16 service framework code.
