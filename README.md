# ML_F16C Community Tutorial

Community release repository for F-16C cockpit simulator panels, gauges, indicators, and supporting build notes for DCS World.

This repository contains simplified community/tutorial versions of selected `ML_F16` cockpit simulator modules. The aim is to help other cockpit builders understand the basic hardware, wiring, display logic, and DCS-BIOS integration patterns used in small F-16C simulator panels.

## Current Community Release

The first community release is the:

# F-16C Fuel Flow Indicator Community Edition

Development branch:

`ML_F16_FFI_Community`

Release tag:

`v1.0-ffi-community`

This release demonstrates a simplified Arduino Nano + OLED based Fuel Flow Indicator using DCS-BIOS for DCS World.

## Project Status

The repository is currently in its first public community/tutorial phase.

The focus of this release is to provide a working, understandable, and buildable example rather than a fully commercial cockpit product.

Advanced prototype, commercial, service, hidden diagnostics, calibration, enclosure, and protected development features are not included in the community edition.

## Community vs Advanced Builds

The community releases are intended for learning, experimentation, and hobby cockpit building.

They may include:

- simplified code
- basic DCS-BIOS integration
- basic display rendering
- simple test modes
- introductory build notes

They do not include the full advanced prototype feature set used in private development builds, such as:

- advanced diagnostics
- hidden service modes
- calibration tables
- commercial support tooling
- advanced enclosure revisions
- protected rendering or realism logic
- production-ready documentation packs
